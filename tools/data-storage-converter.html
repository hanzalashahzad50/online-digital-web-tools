  • 1 KiB = 1,024 bytes (binary)
  • 1 GB = 1 billion bytes

About Data Storage Units
Decimal Units (SI):
  • Kilobyte (KB): 1,000 bytes
  • Megabyte (MB): 1,000 kilobytes
  • Gigabyte (GB): 1,000 megabytes
  • Terabyte (TB): 1,000 gigabytes
  • Petabyte (PB): 1,000 terabytes
Binary Units (IEC):
  • Kibibyte (KiB): 1,024 bytes
  • Mebibyte (MiB): 1,024 kibibytes
  • Gibibyte (GiB): 1,024 mebibytes
  • Tebibyte (TiB): 1,024 gibibytes
  • Pebibyte (PiB): 1,024 tebibytes
Usage by Context:
  • Storage Manufacturers: Use decimal (1000-based)
  • Operating Systems: Use binary (1024-based)
  • Network Speeds: Use decimal (1000-based)
  • Memory: Use binary (1024-based)
