Usage Tip: Use this Base64 string directly in your HTML img tag's src attribute or in your CSS background-image property.
How to Use
  1. Click "Select Image" or drag and drop an image file
  2. The image will be converted to Base64 automatically
  3. Choose your preferred output format (Data URL or Base64 only)
  4. Click "Copy" to copy the result to clipboard or "Download" to save as a text file
Example Usage:
<!-- In HTML -->
<img src="data:image/png;base64,YOUR_BASE64_STRING">

/* In CSS */
.background {
    background-image: url('data:image/png;base64,YOUR_BASE64_STRING');
}
Supported Formats

  • PNG
  • GIF
  • WebP
  • BMP
  • SVG
